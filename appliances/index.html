要做到真正实时，建议这样落地

前端不能稳定直接抓取零售网站实时价格，因为会遇到 CORS、反爬、动态渲染和条款限制。更稳的产品形态是：网站负责展示和决策，后端每 15-60 分钟通过授权 feed、联盟 feed、Amazon PA-API 或合规采集更新一次价格。

  1. 第一阶段人工维护 20-50 个常买型号，验证你自己购物时是否真的省钱。
  2. 第二阶段接入 Amazon Product Advertising API 和零售商 / 联盟产品 feed。
  3. 第三阶段加入价格提醒、历史曲线、浏览器采集入口和自动刷新任务。
